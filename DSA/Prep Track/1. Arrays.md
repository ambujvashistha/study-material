# 1 - Arrays

**Arrays in Python**

Python doesn’t have *traditional fixed-size arrays* like C or C++.

Instead, it provides **lists** (most commonly used) and the **array** module (for numeric data).

We’ll see both below.

---

### **1. Declaration and Initialization**

### Using Lists:

```python
arr = [1, 2, 3, 4, 5]      # Initializes a list with 5 elements
arr = [0] * 5              # Initializes all elements to 0
arr = [1] + [0] * 4        # First element 1, rest 0
n = len(arr)               # Number of elements in the list

```

### Using `array` Module:

```python
from array import array
arr = array('i', [1, 2, 3, 4, 5])  # 'i' means integer type

```

Unlike C++, Python lists can grow or shrink dynamically —

so there’s **no need to predefine the size**.

---

### **2. Traversal**

### **Using `for` loop with index:**

```python
for i in range(len(arr)):
    print(arr[i])

```

### **Using for-each loop (value):**

```python
for x in arr:
    print(x)

```

### **Using enumerate() (index + value):**

```python
for i, x in enumerate(arr):
    print(f"Index {i} → Value {x}")

```

---

### **3. Lists vs References (Pointers Analogy)**

Python doesn’t have **pointers**,

but **variables hold references** to objects in memory.

```python
arr = [1, 2, 3, 4, 5, 6]
ptr = arr       # Both refer to the same list

print(arr is ptr)   # True → both names point to the same memory object

```

### Modifying via one reference affects the other:

```python
ptr[2] = 99
print(arr)   # [1, 2, 99, 4, 5, 6]

```

In Python:

- `arr` and `ptr` both point to the same object.
- You **can’t use pointer arithmetic** (like `ptr + 2`) as in C++.

---

### **4. Array Size and Memory Info**

```python
import sys

arr = [1, 2, 3, 4, 5, 6]
print(len(arr))         # Number of elements
print(sys.getsizeof(arr))  # Memory size (bytes)

```

Unlike `sizeof()` in C++, this gives **approximate memory usage**, not total element bytes.

---

### **5. Types of Arrays / Lists**

- **Static-like Arrays** (fixed type, using `array` module):
    
    ```python
    from array import array
    arr = array('i', [1, 2, 3, 4])
    
    ```
    
- **Dynamic Arrays** (using Lists):
    
    ```python
    v = []               # Empty list
    v.append(10)
    v.append(20)
    print(v)             # [10, 20]
    
    ```
    
- **Multidimensional Arrays** (2D list):
    
    ```python
    arr2d = [[1, 2], [3, 4], [5, 6]]
    print(arr2d[1][0])   # 3
    
    ```
    
- **Using NumPy Arrays (for performance):**
    
    ```python
    import numpy as np
    arr = np.array([1, 2, 3, 4, 5])
    print(arr * 2)       # Vectorized operation → [2 4 6 8 10]
    
    ```
