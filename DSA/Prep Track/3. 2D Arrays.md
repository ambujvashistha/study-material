### **2D List Initialization**

- `arr = [[10, 20], [30, 40], [50, 60]]`
    
    A **2D list** (Python’s equivalent of a 2D array) where each sublist represents a row.
    

---

### **Contiguous Storage**

- In Python, lists are **not stored in contiguous memory** like C arrays.
    
    Instead, each list stores **references (pointers)** to objects in memory.
    
    This makes lists more flexible, but slightly slower for numerical computations.
    

---

### **Alternative Initialization**

- `arr = [ [10, 20], [30, 40], [50, 60] ]`
    
    You can also write the same array in a single line — spacing doesn’t matter.
    

---

### **Automatic Size Handling**

- `arr = [[1, 2], [3, 4]]`
    
    Python lists automatically adjust size; you don’t need to declare dimensions.
    

---

### **3D List Initialization**

- `arr = [ [[1, 2], [3, 4]], [[5, 6], [7, 8]] ]`
    
    A **3D list** where:
    
    - There are **2 blocks**,
    - Each block contains a **2×2 matrix**.

---

### **Traversal**

### **Using Nested For Loops**

```python
arr = [[10, 20], [30, 40], [50, 60]]

for i in range(len(arr)):
    for j in range(len(arr[i])):
        print(arr[i][j], end=" ")
    print()


```
### **Using Pythonic Loop (for-each style)**

```python
for row in arr:
    for x in row:
        print(x, end=" ")
    print()

```
